# **App Name**: AccessoryAce

## Core Features:

- Model Compatibility Search: Allow users to search for compatible accessories based on phone brand and model. Prioritize exact match results. Leverage fuzzy search and an LLM tool for inexact matches. Recommend a follow-up question if no results are found.
- Contributor Data Submission: Enable users to submit compatibility data for new phone models or accessories.
- Admin Excel Upload: Provide an admin interface for uploading Excel sheets containing compatibility data.
- Contributor Leaderboard: Implement a public leaderboard that showcases top contributors, sorted by the number of contributions.
- Report Incorrect Data: Allow users to report incorrect compatibility data for review and correction by admins.

## Style Guidelines:

- Primary color: #0B84FF (Trust blue) for a professional and trustworthy feel.
- Accent color: #FF7A59 (Warm contrast) for calls to action and highlights.
- Background color: #FFFFFF (White) for a clean and modern look.
- Headline font: 'Inter' (sans-serif) with a 600-700 weight for headings to convey a modern and bold aesthetic. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) with a 400-500 weight for body text, ensuring readability and consistency. Note: currently only Google Fonts are supported.
- Line + filled icons from Lucide or Feather Icons for a clean and consistent visual language.
- Mobile-first layout with a fixed bottom navigation bar for easy access to core features.
- Subtle animations, such as a slide-up + fade for the results card reveal, to enhance the user experience without being distracting.
